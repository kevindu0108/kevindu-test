The **results** folder is where you will output all the results (tables, figures, numbers) derived by the code in this repository.

- It is ***normal and expected*** that you don't see any files here other than this README.

    - The files you generate are not in version control: your code is committed to the Github repository, but the results are not committed to your Github repository.
    
    - The results are generated from scratch in Codespaces or Github Actions each time your code is executed.

- You can check your results and download a ZIP file containing the results generated by your code by following [these instructions](https://github.com/UofT-Econ-DataAnalytics/files/wiki/Submissions).
